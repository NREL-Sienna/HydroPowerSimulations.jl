Operation Problem with HydroPowerSimulations.jl

Originally Contributed by: Rodrigo Henriquez-Auba

Load packages:

julia> using PowerSystems
julia> using PowerSimulations
julia> using HydroPowerSimulations
julia> using PowerSystemCaseBuilderERROR: ArgumentError: Package PowerSystemCaseBuilder not found in current path.
- Run `import Pkg; Pkg.add("PowerSystemCaseBuilder")` to install the PowerSystemCaseBuilder package.
julia> using HiGHS # solver
